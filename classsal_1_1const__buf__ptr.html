Detailed Description

const_buf_ptr represents contiguous region of raw memory with specified size.

Internally it is kept as pointer pair to beginning and one byte past end of region. Object of this class does not own pointed region. It is application responsibility to handle lifecycle of buf_ptr and pointed region.
