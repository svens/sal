Detailed Description

template<typename T, typename SyncPolicy>
class sal::queue_t< T, SyncPolicy >

Queue (FIFO) with possible concurrent usage.

This class provides methods to push elements on the back of queue and pops them from front. Default implementation provides no synchronisation guarantees i.e. it is undefined behaviour to access it from multiple threads without explicit external locking.

Specify one of synchronisation policies from sal/sync_policy.hpp to get synchronised queue implementation. Currently only sal::spsc_sync_t is implemented.
