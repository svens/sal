Detailed Description

template<typename T, typename Mutex>
class sal::locked_ptr< T, Mutex >

Scoped locking idiom extension to hold pointer to data during locked scope.

Once unlocked, data pointer is reset also. It is undefined behaviour to store pointer to external variable and continue using is after this locked_ptr is unlocked.

Usage:

using lockable_int = sal::lockable_t<int>;
int data;
lockable_int lockable_data{data};
if (auto data_lock = lockable_data.lock())
{
*data_lock = 1;
}
