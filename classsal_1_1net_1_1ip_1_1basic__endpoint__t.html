Detailed Description

template<typename Protocol>
class sal::net::ip::basic_endpoint_t< Protocol >

Endpoint represents a protocol-specific endpoint.

It consists of an IP address and port number. Endpoints are used to identify sources and destinations for socket connections and datagrams.
