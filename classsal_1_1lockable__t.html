Detailed Description

template<typename T, typename Mutex = std::mutex>
class sal::lockable_t< T, Mutex >

Extended Lockable concept.

This class internally owns mutable Mutex and reference to asscociated data of type T. To access associated data, call any of locking methods or for explicitly unsynchronised access, use unlocked() method to acquire locked_ptr.
