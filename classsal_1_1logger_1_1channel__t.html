Detailed Description

template<typename Worker>
class sal::logger::channel_t< Worker >

Main event logging API.

It provides only limited methods to check if logging is enabled and event factory method. This API is intentionally limited to separate channel creation and configuring from logging events.

Worker owns channel_t instances and manages their lifecycle. Possible implementations provided by this library are worker_t (synchronous logging) and async_worker_t (asynchronous logging). This class has internally only reference to actual channel. This way it is cheap to copy and pass around.

It is not recommended to use this class' methods directly but through logging macros that can be configured compile time.
