Detailed Description

template<typename Algorithm>
class sal::crypto::hash_t< Algorithm >

One way cryptographic hashing using Algorithm.

If full data is already in continuous memory region, call static one_shot() to calculate digest immediately.

If data is not completely stored in memory, instantiate this class and add new data in one or more calls to update(). Once full data has been fed into hasher, call finish() to fetch calculated digest. Application can immediately reuse instance after call to finish(), it is re-initialized internally.
