Detailed Description

template<typename AcceptableProtocol>
class sal::net::basic_socket_acceptor_t< AcceptableProtocol >

Object of class basic_socket_acceptor_t is used to listen and queue incoming socket connections.

Socket object that represent incoming connections are dequeued by calling accept().

For more information about asynchronous API usage, see io_service_t.
