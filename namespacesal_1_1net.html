◆ init()

const std::error_code& sal::net::init ( )
inlinenoexcept

On Windows platform, initialize winsock library.

There is no need to call it explicitly as it is also done internally by static initialization. Only exception is when application layer own static initialization order depends on winsock library to be already loaded. It can be called multiple times. std::call_once is used to make sure only single call proceeds.
